## 3. Calm UX 디자인 원칙

### 3.1 핵심 철학

```
┌─────────────────────────────────────────────────────────────┐
│                    CALM UX PHILOSOPHY                        │
│               "앱이 스트레스가 되어선 안 된다"                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1️⃣ 부드러움 (Gentle)                                       │
│     • 절대 사용자를 다그치지 않는다                         │
│     • 실패해도 격려하는 메시지                              │
│     • 부드러운 애니메이션과 전환                            │
│                                                             │
│  2️⃣ 단순함 (Simple)                                         │
│     • "오늘"에만 집중하는 기본 뷰                           │
│     • 3-tap 이내로 모든 기능 접근                           │
│     • 불필요한 옵션 최소화                                  │
│                                                             │
│  3️⃣ 자율성 (Autonomy)                                       │
│     • 강제하지 않는 알림                                    │
│     • 스킵 가능한 모든 단계                                 │
│     • 사용자 페이스 존중                                    │
│                                                             │
│  4️⃣ 따뜻함 (Warmth)                                         │
│     • 기계적이지 않은 AI 톤앤매너                           │
│     • 캐릭터를 통한 정서적 연결                             │
│     • 인간적인 격려와 응원                                  │
│                                                             │
│  5️⃣ 즐거움 (Joy)                                            │
│     • 작은 성취에도 시각적 보상                             │
│     • 예상치 못한 귀여운 순간들                             │
│     • 사용할수록 풍성해지는 경험                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 Do's and Don'ts

| DO ✅ | DON'T ❌ |
|-------|---------|
| "오늘도 잘 하고 있어요!" | "아직 3개 안 했어요!" |
| "쉬어가도 괜찮아요" | "연속 기록이 끊겼습니다" |
| "가볍게 시작해볼까요?" | "할 일이 5개 밀렸습니다" |
| "다음에 또 만나요 :)" | "앱을 열지 않은 지 3일..." |
